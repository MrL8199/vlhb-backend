# Online-Bookstore

## Overview
An ecommerce book-shop web app developed with Dinh Van Luc, Pham Trung Hieu, Tran Thai Bao - student at Le Quy Don Technical University, written in Python using Flask. 
Functionalities include login, shopping cart, searching, adding and updating books, file upload (book image), and identity authentication,..etc

## Running the project
- Create new environment for project (conda, virtual env,..)
- Install package requirements using:
```commandline
pip install -r requirements.txt
```
- Migrate the database into `MySQL`
- Open the Flask project in `Pycharm/Visual Studio Code`. Click run (on development server)
